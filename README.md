# Daily Market Note
매일 경제와 증권 시장을 공부하기 위한 학습 노트 자동화 프로젝트입니다.

## 목적

이 프로젝트는 금융 도메인 이해도를 높이기 위해 시작했습니다.

매일 주요 경제와 증권 뉴스 및 시장 이슈를 수집하고, AI를 활용해 학습노트 초안을 생성한 뒤 직접 해석을 추가해 기록하는 것을 목표로 합니다.

## 자동화 흐름

1. 주요 경제·증권 뉴스 수집
2. 관심 키워드 기반 분류
3. AI 기반 학습노트 초안 생성
4. Markdown 파일 저장
5. GitHub Actions를 통한 평일 자동 실행

## 주의사항

본 프로젝트의 글은 개인 학습 기록이며, 특정 종목에 대한 투자 권유가 아닙니다.
